## User Story
Como usuário do Aura visualizando a rede PERT, quero que os nós sejam retângulos arredondados compactos exibindo ID e duração, para que mais informação caiba na viewport e o diagrama seja mais profissional.

## Acceptance Criteria
- [x] AC-1: Nós são `<rect>` arredondados (rx=6) em vez de `<circle>`
- [x] AC-2: Dimensões: w=64px, h=36px (compacto)
- [x] AC-3: Linha superior: ID da tarefa (display code), fonte monospace bold
- [x] AC-4: Linha inferior: duração em dias (ex: "5d"), fonte menor
- [x] AC-5: Borda vermelha (#f43f5e) para caminho crítico primário, laranja para secundário, slate para folga
- [x] AC-6: Tooltip ao hover: nome completo + ES/EF/LS/LF/Folga
- [x] AC-7: TypeCheck: 0 erros
