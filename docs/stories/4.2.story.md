# Story 4.2 — Layout Sugiyama: Barycenter Heuristic para Redução de Cruzamentos
**Épico:** EP-04 Diagramas PERT e Gantt
**Sprint:** PERT-V2
**Status:** Done
**Agentes:** @dev (Dex), @dataviz (Viz)

## User Story
Como usuário do Aura visualizando a rede PERT, quero que os nós sejam posicionados usando o algoritmo Sugiyama com barycenter heuristic, para que cruzamentos de arestas sejam minimizados e o diagrama seja legível em projetos com até 50 tarefas.

## Acceptance Criteria
- [x] AC-1: Layout usa barycenter heuristic (2 passes: top-down + bottom-up) para ordenar nós dentro de cada camada
- [x] AC-2: Nós dentro da mesma camada são centralizados verticalmente em relação à janela total
- [x] AC-3: Nós em camadas com apenas 1 elemento ficam alinhados ao centroide vertical dos seus sucessores/predecessores
- [x] AC-4: Layout encapsulado em `computeSugiyamaLayout(tarefas): Map<string, {cx, cy}>` exportado de `src/lib/engine/sugiyama.ts`
- [x] AC-5: TypeCheck: 0 erros
