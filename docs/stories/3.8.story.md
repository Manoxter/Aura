## User Story
Como sistema de qualidade do Aura,
quero um A/B test automatizado que compara o novo modelo calibrado (σ por setor + NVO atualizado) contra o modelo anterior,
para que o modelo com menor erro no caso Big Dig se torne o padrão automaticamente.

## Background
- Big Dig = Boston Central Artery Project: projeção de custo $2.6B, realizado $14.6B (custo/setor = infraestrutura)
- Critério de vitória: menor RMSE em relação ao desfecho real do Big Dig
- Gate K1 do INSIGHTS-LOG: validação com caso histórico real antes de escalar
