## User Story
Como motor matemático do Aura,
quero que os lados C e P do triângulo CDT sejam derivados de tangentes pontuais instantâneas (não de regressão OLS global),
para que o triângulo reflita o estado real do projeto no instante atual, não uma média histórica.

---

- [x] AC-3: No dia 0 (baseline): C=1, P=1 → triângulo equilátero
- [x] AC-4: Função `tangentePontual()` exportada de `math.ts` (usável em testes)
