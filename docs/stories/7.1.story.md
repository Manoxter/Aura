## User Story
Como PM usando o Aura,
quero descrever uma decisão gerencial em texto livre (ex: "Aprovei um aditivo de 20% no orçamento e postergamos 2 sprints")
e receber automaticamente os parâmetros MATED estruturados com a zona de impacto no triângulo CDT,
para que a decisão fique registrada com contexto analítico sem precisar preencher formulários numéricos.

## Notas Técnicas
- `impacto_area_percent` calculado como média dos deltas absolutos (|E|+|P|+|O|)/3 × 100
- Fallback de `confianca` = 0.5 e `justificativa` = `''` quando ausentes na resposta LLM
- Parser tenta 3 estratégias de extração JSON: puro, markdown code block, substring entre `{}`
- Persistência em `decisoes_mated` falha silenciosa: logError sem bloquear resposta ao cliente
- temperatura Groq = 0.1 (mais determinístico que o klauss chat = 0.2)
