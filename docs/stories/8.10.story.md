## User Story
Como engenheiro de dados,
quero que ao arquivar um projeto uma Edge Function registre automaticamente um evento empírico de calibração Bayesiana,
para que o modelo de predição do Aura seja refinado com dados reais de projetos concluídos.

---

## Acceptance Criteria
- [x] AC-1: Edge Function `calibrate-on-archive` existe e está deployada na Supabase.
- [x] AC-2: A função é disparada ao arquivar um projeto (trigger via webhook ou chamada direta).
- [x] AC-3: Atualiza a tabela `aura_calibration_events` com os dados empíricos do projeto arquivado.
- [x] AC-4: Falha na Edge Function não bloqueia o arquivamento — comportamento de falha silenciosa implementado.
