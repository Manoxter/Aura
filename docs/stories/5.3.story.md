## User Story
Como motor matemático do Aura,
quero que o MATED seja recalculado automaticamente após cada atualização do TA,
para que o Índice de Qualidade (IQ) e o painel MATED reflitam o estado real em tempo real.

---

## Acceptance Criteria

### AC-1 — Chamada de calcularMATED() após recalcularTA()
- [ ] recalcularTA() chama calcularMATED(novoTA, TM) após TA válido calculado
- [ ] Chamada é fire-and-forget (não bloqueia saveProgresso())
- [ ] Se TM não disponível: calcularMATED() não é chamado — guarda silenciosamente

### AC-2 — Resultado disponível no contexto
- [ ] ProjectContext expõe matedAtual: number | null atualizado após cada recálculo
- [ ] matedAtual = null se TM ainda não gerado (projeto novo sem baseline)

### AC-3 — IQ atualizado
- [ ] Índice de Qualidade (badge TA/TM) reflete o novo matedAtual sem reload manual

### AC-4 — Sem regressão
- [ ] Comportamento de recalcularTA() inalterado para E, P, O, CEt
- [ ] TypeCheck: 0 erros novos
- [ ] Todos os testes de execution.test.ts passam
