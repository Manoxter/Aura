# Story 8.7 — localStorage: expiração 24h + limpeza ao logout
**Épico:** EP-08 Infra/DB
**Sprint:** F-CICD
**Status:** Done
**Agentes:** @security-auditor, @dev (Dex)

---

## User Story
Como engenheiro de segurança,
quero que dados salvos no localStorage expirem automaticamente em 24h e sejam limpos no logout,
para que informações sensíveis de sessão não persistam indefinidamente no navegador do usuário.

---

## Acceptance Criteria
- [x] AC-1: `setLocalStorage(key, value, ttlHours=24)` salva o valor com timestamp de expiração embutido.
- [x] AC-2: `getLocalStorage(key)` retorna `null` e remove a chave se o TTL expirou.
- [x] AC-3: Ao fazer logout, todas as chaves com prefixo `aura_` são removidas do localStorage.
- [x] AC-4: Testes unitários cobrem: set/get normal, get expirado, get inexistente e logout cleanup.
- [x] AC-5: TypeCheck limpo — 0 erros novos introduzidos.
