## User Story
Como engenheiro DevOps,
quero que o merge em `main` dispare automaticamente o deploy na Vercel,
para que toda entrega aprovada seja publicada sem intervenção manual.

---

- [x] AC-1: Deploy automático é disparado ao fazer merge em `main`.
- [x] AC-2: Preview deploy é gerado para cada PR aberto.
- [x] AC-3: URL do deploy (production ou preview) aparece como comentário no PR.
