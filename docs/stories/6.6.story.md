**Status:** Done
**Agentes:** @dev (Dex), @ux-design-expert (Uma)

## User Story
Como PM usando o Aura pela primeira vez,
quero que cada seção do pipeline (TAP, WBS, Tarefas, Motor) exiba uma mensagem de empty state específica e orientadora,
para que eu saiba exatamente o que fazer para desbloquear a próxima etapa.

---

## Acceptance Criteria
- [ ] AC-1: TAP vazio: mensagem orientando a preencher o Termo de Abertura do Projeto
- [ ] AC-2: WBS vazio (mas TAP preenchido): mensagem orientando a criar a estrutura analítica
- [ ] AC-3: Tarefas vazia: mensagem com CTA para adicionar tarefas a partir da WBS
- [ ] AC-4: Motor bloqueado (SETUP < 60%): mensagem com % atual e o que falta para desbloqueio
- [ ] AC-5: Governança bloqueada: mensagem indicando pré-requisitos pendentes
- [ ] AC-6: Empty states têm ícone, título, descrição e botão de ação (CTA)
- [ ] AC-7: TypeCheck: 0 erros
