# Story 4.6 — GanttLupa: Overlay Hover ±15% sobre Função Custo
**Épico:** EP-04 Diagramas PERT e Gantt
**Sprint:** GANTT-LUPA
**Status:** Done
**Agentes:** @dataviz (Viz), @dev (Dex)

## User Story
Como PM no Aura visualizando a Função Custo, quero que ao passar o mouse sobre o gráfico apareça um overlay tipo lupa mostrando o Gantt da janela temporal ±15% ao redor do ponto hovered, para correlacionar custo e cronograma sem trocar de aba.

## Acceptance Criteria
- [x] AC-1: Abaixo da CostChart (tab Custos) existe um eixo temporal fino como linha de base — sem barras visíveis em estado normal
- [x] AC-2: Ao mover o mouse sobre a CostChart, o `hoveredDay` é calculado: tarefa mais próxima do cursor → midpoint (ES+EF)/2
- [x] AC-3: Um overlay flutuante (280px de largura) exibe barras do Gantt apenas para tarefas ativas em [hoveredDay×0.85, hoveredDay×1.15]
- [x] AC-4: Overlay posicionado logo abaixo do cursor, dentro do container da seção
- [x] AC-5: Ao sair do cursor do gráfico, overlay desaparece (a menos que fixado — Story 4.8)
- [x] AC-6: TypeCheck: 0 erros
