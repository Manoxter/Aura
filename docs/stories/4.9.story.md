## User Story
Como PM em projetos com prazos muito distintos (1 mês a 5 anos), quero que o eixo temporal da lupa e do Gantt exiba a unidade de tempo mais legível (dias, semanas, meses ou anos), com labels inline que não se sobreponham.

## Acceptance Criteria
- [x] AC-1: Regra de seleção de unidade: maxEF ≤60 → dias; ≤400 → semanas; ≤1200 → meses; >1200 → anos
- [x] AC-2: Eixo base do GanttLupa exibe 5 ticks com labels na unidade selecionada
- [x] AC-3: Header do overlay exibe a posição na unidade correta (ex: "Semana 12" em vez de "Dia 84")
- [x] AC-4: Label inline: não sobrepõe o tick anterior (mínimo 40px de espaçamento entre labels)
- [x] AC-5: TypeCheck: 0 erros
