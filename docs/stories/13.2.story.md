# Story 13.2 — UI: tela Riscos — lista, filtros e CRUD
**Épico:** EP-13 Prometeu Extrínseco

**Status:** Done
**Agentes:** @dev, @ux-design-expert
**Pré-requisito:** 13.1 Done

## User Story
Como PM do projeto,
quero uma tela dedicada para gerenciar os riscos do projeto com lista, filtros por categoria e operações de criação e exclusão,
para que eu possa monitorar e atualizar o mapa de riscos de forma centralizada.

---

## Acceptance Criteria
- [x] AC-1: Página criada em `/[projetoId]/governanca/riscos/` acessível via menu de governança.
- [x] AC-2: Lista de riscos exibe todos os riscos do projeto com filtro funcional por categoria (`escopo`, `prazo`, `custo`, `qualidade`, `externo`).
- [x] AC-3: Botão "Novo Risco" abre modal de criação com campos: título, categoria, probabilidade (0-1), impacto (0-1) e descrição opcional.
- [x] AC-4: Badge de `score_rc` exibido em cada item com cores: verde se `< 0.3`, amarelo se `0.3–0.6`, vermelho se `> 0.6`.
- [x] AC-5: Ação de exclusão com diálogo de confirmação antes de deletar o risco.
- [x] AC-6: TypeCheck limpo — 0 erros novos introduzidos.
