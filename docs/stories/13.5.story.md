## User Story
Como motor matemático do Aura,
quero uma função pura `calcularScoreRC(probabilidade, impacto)` que computa o Score de Risco Composto,
para que o Prometeu Extrínseco possa quantificar cada risco de forma padronizada e testável.

---

## Background Técnico
Score RC = w₁·P + w₂·I + w₃·(P × I)
Onde:
- P = probabilidade ∈ [0, 1]
- I = impacto ∈ [0, 1]
- w₁ = 0.3, w₂ = 0.3, w₃ = 0.4 (pesos padrão Aura)

Faixas de classificação:
- Verde: score_rc < 0.3
- Amarelo: 0.3 ≤ score_rc ≤ 0.6
- Vermelho: score_rc > 0.6
