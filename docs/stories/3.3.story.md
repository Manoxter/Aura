## User Story
Como motor de calibração do Aura,
quero fatores de regressão ponderada específicos por setor,
para que a curva de atualização Bayesiana use o peso correto ao acumular dados empíricos.

## Background
Regressão Ponderada: `Peso_i = (i+1)/n × fator_atividade`
Onde `fator_atividade` varia por setor:
- Construção Civil: 1.6 (alta variabilidade histórica)
- Tecnologia: 1.2 (mais previsível, mas scope creep frequente)
- Infraestrutura: 1.4 (variabilidade média)
- Saúde: 1.0 (alta regulação, menor variabilidade)
