## User Story
Como PM usando o Aura,
quero poder clicar diretamente no canvas do Triângulo Matriz nos boards de Orçamento e Prazo para definir a posição do ponto TA interativamente,
para que eu possa ajustar parâmetros de forma mais intuitiva do que via inputs numéricos.
