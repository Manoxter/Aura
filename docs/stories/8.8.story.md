## User Story
Como engenheiro DevOps,
quero que todo PR aberto tenha lint, typecheck e vitest executados automaticamente via GitHub Actions,
para que código com erros não seja mergeado na branch principal.

---

## Acceptance Criteria
- [x] AC-1: Arquivo `.github/workflows/ci.yml` existe no repositório.
- [x] AC-2: Steps de lint, typecheck e vitest são executados automaticamente em cada PR.
- [x] AC-3: Merge é bloqueado se qualquer um dos três steps falhar.
