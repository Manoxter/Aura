## User Story
Como engenheiro de dados,
quero que o `score_rc` seja calculado e persistido automaticamente ao criar ou atualizar um risco,
para que a coluna `score_rc` na tabela `riscos_projeto` reflita sempre o valor atual da fórmula.

---

## Acceptance Criteria
- [x] AC-1: A API route POST de riscos (Story 13.3) calcula `score_rc` via `calcularScoreRC()` antes de inserir.
- [x] AC-2: O valor de `score_rc` é persistido na coluna `riscos_projeto.score_rc`.
- [x] AC-3: Ao exibir a lista de riscos (Story 13.2), o `score_rc` lido do banco é o mesmo calculado pela função.
- [x] AC-4: Nenhum trigger de banco é necessário — toda lógica de cálculo é na camada de aplicação.
- [x] AC-5: TypeCheck limpo — 0 erros novos introduzidos.
