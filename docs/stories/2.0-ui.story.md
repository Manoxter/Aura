## User Story

Como PM usando o Aura,
quero visualizar os ângulos α, ω, ε do triângulo de monitoramento em tempo real através do Painel Integridade do Triângulo,
para que eu possa avaliar a saúde angular do projeto de forma intuitiva no modo de visualização que mais se adapta ao meu perfil de leitura.

---

## Background

O "Painel Integridade do Triângulo" (nome de UI) expõe os ângulos calculados pela Síntese de Clairaut (Story 2.0-engine).
O usuário escolhe entre 3 modos de visualização com live preview — analogia ao seletor de mira em jogos FPS.

**Nomenclatura:**
- Nome técnico/documentação: **Painel Clairaut**
- Nome de UI (visível ao usuário): **Integridade do Triângulo**

**Três ângulos e suas escalas:**
- **α (Absorção)** — vértice Custo-Escopo: leitura INVERSA (longe de 90° = verde, próximo = vermelho)
- **ω (Entrega)** — vértice Prazo-Escopo: leitura INVERSA
- **ε (Equilíbrio)** — vértice Prazo-Orçamento: leitura DIRETA (próximo de 90° = verde, afastado = vermelho)
